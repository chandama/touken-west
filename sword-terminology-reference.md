# Japanese Sword Terminology Reference
## Comprehensive Guide for Preserving Romanized Terms in Translations

### Blade Structure & Anatomy
#### Main Blade Parts
- **Ha** / **Hasaki** - Edge / cutting edge
- **Shinogi** (鎬) - Ridge line
- **Shinogi-ji** (鎬地) - Surface between ridge and back
- **Ji** / **Jigane** / **Jihada** (地/地鉄/地肌) - Blade surface/ground/steel surface
- **Hira-ji** (平地) - Flat surface area
- **Mune** (棟) - Back/spine of blade
- **Iori-mune** (庵棟) - Ridge-shaped back
- **Mitsu-mune** (三つ棟) - Three-sided back
- **Maru-mune** (丸棟) - Round back
- **Kaku-mune** (角棟) - Square back
- **Monouchi** (物打) - Striking area (upper third of blade)

#### Point/Tip (Kissaki 切先)
- **Kissaki** (切先) - Point/tip section
- **O-kissaki** (大切先) - Large point
- **Chu-kissaki** (中切先) - Medium point
- **Ko-kissaki** (小切先) - Small point
- **Ikari-o-kissaki** (猪首大切先) - Extended large point
- **Boshi** (帽子) - Temper line in the point
- **Yokote** (横手) - Dividing line between blade body and point
- **Fukura** (膨ら) - Curvature of the point edge
- **Fukura-tsuku** (膨ら付く) - With curvature (in point)
- **Fukura-kareru** (膨ら枯れる) - Withered/straight (in point)
- **Ko-maru** (小丸) - Small round point
- **Chu-maru** (中丸) - Medium round point

#### Tang (Nakago 中心)
- **Nakago** (中心) - Tang
- **Ubu-nakago** (生ぶ中心) - Original unaltered tang
- **Suriage** (磨上) - Shortened tang
- **O-suriage** (大磨上) - Greatly shortened
- **Machi** (区) - Notch at blade/tang boundary
- **Hamachi** (刃区) - Notch at edge side
- **Munemachi** (棟区) - Notch at spine side
- **Machi-okuri** (区送り) - Notches moved down (from shortening)
- **Yasurime** (鑢目) - File marks on tang
- **Kiri-yasurime** (切鑢目) - Cross-cut file marks
- **Katte-sagari** (勝手下り) - Diagonal descending file marks
- **Sujikai** (筋違) - Diagonal file marks
- **Ō-sujikai** (大筋違) - Large diagonal file marks
- **Higaki** (檜垣) - Cypress fence pattern file marks
- **Keshō-yasurime** (化粧鑢目) - Decorative file marks
- **Mekugi-ana** (目釘孔) - Peg hole(s)
- **Nakago-jiri** (中心尻) - Tang tip shape
- **Kurijiri** (栗尻) - Chestnut-shaped tip
- **Kengyo** (剣形) - Sword-shaped tip
- **Iriyama-gata** (入山形) - Mountain-entry shape

#### Signature & Inscriptions
- **Mei** (銘) - Signature
- **Omote-mei** (表銘) - Signature on front
- **Ura-mei** (裏銘) - Inscription on back
- **Mumei** (無銘) - Unsigned
- **Ō-mei** (太銘) - Thick/bold characters
- **Hosomei** (細銘) - Thin/fine characters
- **Kinzogan-mei** (金象嵌銘) - Gold inlay signature
- **Ginzogan-mei** (銀象嵌銘) - Silver inlay signature
- **Gakumei** (額銘) - Added signature panel
- **Orikaeshi-mei** (折返銘) - Folded-back signature
- **Kiritsuke-mei** (切付銘) - Cut/chiseled signature
- **Niji-mei** (二字銘) - Two-character signature
- **Nen-ki** (年紀) - Date inscription
- **Saku** (作) - Made by
- **Tsukuru** (造) - Made/crafted

### Tempering Line (Hamon 刃文) Patterns & Styles

#### Basic Hamon Types
- **Hamon** (刃文) - Temper line
- **Suguha** (直刃) - Straight temper line
- **Notare** (湾れ) - Gentle undulating pattern
- **O-notare** (大湾れ) - Large notare
- **Ko-notare** (小湾れ) - Small notare
- **Chu-notare** (中湾れ) - Medium notare
- **Gunome** (互の目) - Clove/irregular pattern
- **O-gunome** (大互の目) - Large gunome
- **Ko-gunome** (小互の目) - Small gunome
- **Chu-gunome** (中互の目) - Medium gunome
- **Midare** (乱れ) - Irregular/active pattern
- **O-midare** (大乱れ) - Large irregular pattern
- **Ko-midare** (小乱れ) - Small irregular pattern
- **Choji-midare** (丁子乱れ) - Clove-shaped irregular pattern
- **O-choji** (大丁子) - Large clove pattern
- **Ko-choji** (小丁子) - Small clove pattern

#### Complex Hamon Patterns
- **Hitatsura** (皆焼) - All-over tempering covering ji
- **Togari** (尖り) - Pointed pattern
- **Togari-ba** (尖り刃) - Pointed edge pattern
- **Yahazu** (矢筈) - Arrowhead notch pattern
- **Sanbon-sugi** (三本杉) - Three-cedar pattern
- **Kawazuko-choji** (蛙子丁子) - Tadpole clove pattern
- **Juka-choji** (重花丁子) - Layered flower clove pattern
- **Kyo-gunome** (京互の目) - Kyoto-style gunome
- **Koshu-notare** (甲州湾れ) - Koshu province style notare
- **Bizen-choji** (備前丁子) - Bizen province choji style
- **Toran** (濤瀾/涛乱) - Large swirling waves
- **Saka-choji** (逆丁子) - Inverted clove pattern
- **Kakko-choji** (角丁子) - Angular clove pattern
- **Tsukare** (疲れ) - Fatigued/weakened pattern

#### Hamon Activity & Features
- **Nie** (沸) - Coarse bright crystals (martensite)
- **Nioi** (匂) - Fine misty crystals
- **Nie-deki** (沸出来) - Nie-based/prominent nie work
- **Nioi-deki** (匂出来) - Nioi-based work
- **Ara-nie** (荒沸) - Coarse nie
- **Ko-nie** (小沸) - Fine nie
- **Ji-nie** (地沸) - Nie in the ji surface
- **Nioi-guchi** (匂口) - Nioi edge/boundary of hamon
- **Hotsure** (ほつれ) - Fraying/breaking up of hamon
- **Sunagashi** (砂流し) - Sand flow (streaks in hamon)
- **Kinsuji** (金筋) - Lightning/golden streaks
- **Inazuma** (稲妻) - Lightning bolt effect
- **Yubashiri** (湯走り) - Running (activity in hamon)
- **Ashi** (足) - Legs extending from hamon into ji
- **You** / **Yo** (葉) - Floating islands in hamon
- **Tobiyaki** (飛焼) - Isolated tempering spots
- **Uchinoke** (打ち除け) - Hammered appearance
- **Muneyaki** (棟焼) - Tempering on the back
- **Gyaku-ashi** (逆足) - Reverse legs (extending into ha)
- **Kuichigai-ba** (喰違刃) - Misaligned edge pattern

#### Boshi (Point Tempering 帽子)
- **Kaen** (火炎/火焔) - Flame-shaped
- **Jizo** (地蔵) - Jizo Buddha style (round)
- **Midare-komi** (乱れ込み) - Irregular entering
- **Yakizume** (焼詰) - Temper runs to very tip
- **Yakitsume** (焼き詰め) - Fully tempered to tip
- **Togari** (尖り) - Pointed return
- **Komaru** (小丸) - Small round return
- **O-maru** (大丸) - Large round return
- **Kaeri** (返り) - Turnback/return
- **Kaeri-fukashi** (返り深し) - Deep return
- **Kaeri-asashi** (返り浅し) - Shallow return
- **Nie-kuzure** (沸崩れ) - Nie collapse in point
- **Hakikake** (掃き掛け) - Sweeping pattern
- **Yakizume-fu** (焼詰風) - Yakizume style
- **Ichimai** (一枚) - Single continuous
- **Ko-maru-kaeru** (小丸返る) - Small round returning

### Utsuri (Reflection Effect 映り)
- **Utsuri** (映り/映) - Reflection/shadow effect in ji
- **Bo-utsuri** (棒映り) - Pole/stick utsuri
- **Midare-utsuri** (乱映り) - Irregular utsuri
- **Choji-utsuri** (丁子映り) - Clove-shaped utsuri
- **Nie-utsuri** (沸映り) - Nie-based utsuri

### Grain Pattern (Jihada 地肌)

#### Basic Grain Types
- **Jihada** (地肌) - Surface grain pattern
- **Itame** (板目) - Wood grain pattern
- **O-itame** (大板目) - Large itame
- **Ko-itame** (小板目) - Fine/small itame
- **Mokume** (杢目) - Burl/swirling grain
- **O-mokume** (大杢目) - Large mokume
- **Ko-mokume** (小杢目) - Fine mokume
- **Masame** (柾目) - Straight grain
- **O-masame** (大柾目) - Large straight grain
- **Nashiji** (梨子地) - Pear skin texture
- **Muji** (無地) - Plain/no visible grain
- **Ayasugi-hada** (綾杉肌) - Cedar brushed grain (Gassan style)

#### Grain Characteristics & Features
- **Nagare** (流れ) - Running/flowing grain
- **O-hada** (大肌) - Large/coarse grain
- **Ko-hada** (小肌) - Fine grain
- **Tsumi-hada** (詰肌) - Tight/compact grain
- **Yure** (ゆれ) - Wavering grain
- **Masame-gakari** (柾目がかり) - Tendency toward straight grain
- **Itame-nagare** (板目流れ) - Flowing itame
- **Mokume-itame** (杢目板目) - Mixed burl and wood grain
- **Itame-mokume** (板目杢目) - Itame with mokume mixed
- **O-hada-tsuku** (大肌つく) - Has coarse grain
- **Tsumaru** (詰まる) - Tight/compact
- **Yowai** (弱い) - Weak/subdued
- **Tsuyoi** (強い) - Strong/prominent

#### Ji Surface Activity
- **Chikei** (地景) - Landscape effect (dark lines in ji)
- **Kinsuji** (金筋) - Gold/bright lines in ji
- **Sunagashi** (砂流し) - Sand flow in ji
- **Nie-utsuri** (沸映り) - Nie-based reflection
- **Ji-nie** (地沸) - Nie particles throughout ji
- **Jigane** (地鉄) - Steel/ground metal
- **Jiba** (地刃) - Ground and edge (overall)
- **Uzumaki** (渦巻) - Swirl/spiral in grain
- **Shirake** (白け) - Whitish appearance
- **Tsukare** (疲れ) - Fatigued appearance

### Curvature (Sori 反り)
- **Sori** (反り) - Curvature
- **Torii-zori** (鳥居反り) - Shrine gate curve (high center)
- **Koshizori** (腰反り) - Hip/lower curve
- **Sakizori** (先反り) - Tip/forward curve
- **Uchizori** (内反り) - Reverse/inward curve
- **Muzori** (無反り) - No curve/straight
- **Chu-zori** (中反り) - Medium curve
- **Tsuyoi-sori** (強い反り) - Strong/pronounced curve
- **Yowai-sori** (弱い反り) - Weak/gentle curve

### Engravings & Grooves (Horimono 彫物)

#### Groove Types (Hi 樋)
- **Horimono** (彫物) - Engravings/carvings
- **Hi** (樋) - Groove/fuller
- **Bo-hi** (棒樋) - Straight groove
- **Soe-bi** (添樋) - Accompanying groove
- **Futasuji-hi** (二筋樋) - Two-line groove
- **Naginata-hi** (薙刀樋) - Naginata-style groove
- **Koshi-hi** (腰樋) - Short groove near tang
- **Tsure-hi** (連樋) - Paired/accompanying groove
- **Kaki-nagashi** (掻流し) - Flowing carving style
- **Kaki-toshi** (掻通し) - Groove running through to tang

#### Carved Designs
- **Bonji** (梵字) - Sanskrit characters
- **Ken** (剣) - Sword (vajra) carving
- **Kurikara** (倶利伽羅) - Dragon wrapped around sword
- **Ryu** (龍) - Dragon
- **Gomabashi** (護摩箸) - Ritual stick bridge
- **Sanjiko** (三鈷) - Three-pronged vajra
- **Dokko** (独鈷) - Single-pronged vajra
- **Futari-ryu** (二立龍) - Two standing dragons
- **Tsume** (爪) - Claw carving
- **Suken** (素剣) - Plain/unadorned sword carving
- **Shinogi-suken** (鎬素剣) - Ridged plain sword carving

- **Sakikasane** (先重ね) - Thickness at point
- **Kasane** (重ね) - Thickness

### Special Techniques & Characteristics
- **Masame-yaki** (柾目焼) - Tempering following straight grain
- **Mizukage** (水影) - Water shadow effect
- **Uchinoke** (打ちのけ) - Forging marks appearing in hamon
- **Yubashiri** (湯走り) - Activity "running" in hamon
- **Ara-shitate** (粗仕立て) - Rough/bold workmanship
- **Chiisaku-tsumu** (小さく詰む) - Finely compacted
- **Yoku-tsumariru** (良く詰まる) - Well compacted/tight
